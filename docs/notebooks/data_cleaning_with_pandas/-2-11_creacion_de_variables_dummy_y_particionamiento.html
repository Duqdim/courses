Creación de variables dummmy y particionamiento del dataset¶

  • 60:00 min | Última modificación: Agosto 13, 2021 | YouTube

Una tabla gestionada en Pandas puede ser usada como datos para el entrenamiento de modelos en machine learning. En esta lección se abordan problemas específicos de esta tarea.

Al finalizar el documento, usted estará en capacidad de:

  • Crear variables dummmy usando one-hot-encoder.

  • Mezclar un dataset.

  • Crear subconjuntos de datos.

Preparacion¶

[2]:

import pandas as pd

Carga de datos¶

[3]:

iris_flowers = pd.read_csv(
    "https://raw.githubusercontent.com/jdvelasq/datalabs/master/datasets/iris.csv",
    sep = ',',
    thousands = None,
    decimal = '.')

    [

        pd.get_dummies(iris_flowers.Species),
    ],
    axis = 1,
