Fundamentos de pronóstico de series de tiempo¶

  • 34:29 min | Ultima modificación: Marzo 21, 2021 | YouTube

Serie de tiempo. Se define como la secuencia de observaciones:

\[y_0, y_1, y_2,...,y_T\]

Media.

\[\mu_t = E(y_t)\]

Covarianza.

\[\gamma_{t, s} = Cov(y_t, y_s)\]

donde: \(Cov(y_t, y_s) = E(y_t - \mu_t)(y_s - \mu_s) = E(y_t * y_s) - \mu_t \mu_s\)

Función de autocorrelación.

\[\rho_{t, s} = \frac{Cov(y_t, y_s)}{\sqrt{Var(y_t) Var(y_s)}}\]

Paseo aleatorio. Si se define \(y_0 = 0\) y \(y_t = y_{t-1} + e_t\), con \(e_t\) ~ \(N(0, \sigma^2)\):

Estacionalidad. Se define como:

  • La media no cambia sobre el tiempo.

  • La varianza no depende del tiempo.
